Resumen

Este proyecto es un e-commerce enfocado a la venta de vestimenta para personas con gusto y apreciación de lo artesanal.

Su desarrollo se dio durante el bootcamp de Generation Mexico para convertirnos en Desarrolladores Java Full Stack Jr. En el proceso, cada uno de nosotros colaboramos estrechamente bajo el marco de trabajo SCRUM y para el desarrollo Frontend se utilizaron tecnologías web como HTML, CSS y Javascript, con apoyo de Bootstrap.
